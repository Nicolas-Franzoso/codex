# plano-de-midia-make

Skill para gerar plano de midia da Make a partir de transcricao de R1, briefing comercial ou pedido direto do growth.

## Como usar no Codex

Depois de instalar a skill e reiniciar o Codex, voce pode pedir coisas como:

- "Use a skill de plano de midia"
- "Crie um plano de midia para este cliente"
- "Vou colar a transcricao da R1 para montar o plano de midia"

## Como instalar em outra maquina

No Codex, cole:

```text
Use a skill $skill-installer e instale esta skill do GitHub:
https://github.com/Nicolas-Franzoso/codex/tree/main/plano-de-midia-make
```

Depois reinicie o Codex.

## Observacao

Para aparecer automaticamente na lista de skills da sessao, a skill precisa estar instalada localmente em `~/.codex/skills/plano-de-midia-make`.
